## 1. 裁决摘要（agent 已裁，agent 自主记录）

D-131 由**单选 A** 增补为 **混合:compute=A（LLVM DirectX 后端直 emit DXIL）/ 图形=B（MIR→SPIR-V→SPIRV-Cross→HLSL→dxc→DXIL 转译）**。

| 维度 | 路径 | 依据（证据指针） |
|---|---|---|
| compute kernel | **A**（RFC-0003 既有） | 结构首选 + round-8 浅修 PSV（validator pre 0/25→post 25/25 accept，RD-011 dev 偏差） |
| 图形着色阶段 | **B**（本包 RFC-0004） | A 路图形签名 ISG1/OSG1 `elemcount=0` 不可达（`dxil_slice3_rxs0159_sig_disasm_round8.md`，#90504 + FFI ABI 禁区）;A-graphics ~800-1500 LOC 上游大功能无在途（`dxil_a_graphics_sig_effort_report.md`）;B 图形签名 `elemcount>0`、SV 端到端存活、validator accept、确定性 **measured 可行**（`dxil_b_graphics_sig_report.md`） |
| A-graphics | 迁移目标（RD-015） | 上游 #90504/#57928 成熟 + D-205 pin bump 后由 B 迁回 A |

**P-01 达标要求**（agent 裁断，agent 自主记录）:agent 裁定 **P-01(strict-only)不开例外、不设边界**——RFC-0004 §4.4 由「P-01 边界/例外声明占位」改写为 **strict-only 达标要求**:图形=B 严禁任何对用户声明/可观察签名元素的静默降级或丢弃,留不住即显式 6xxx 编译错;经用户语义名 by-construction 保真 + **强制译后签名一致性校验门**(译后 DXIL ISG1/OSG1 vs MIR 意图签名比对 → 6xxx,不可裁剪)达标(RFC-0004 §4.4/§4.2,measured 可行见 `dxil_b_strict_only_report.md`)。细化边界已定稿:「用户可观察」= 外部接口签名元素 + `#[builtin]` 系统值 + 顶点输入/片元输出语义 + 阶段间字段/类型/插值/location 配对;**未用外部输入 = 显式诊断**,不要求强制保留。核心规范句为 agent 自主记录 agent 裁断(agent 合并本决策包生效),不自创 P-01 例外(P-13/硬规则 1)。
